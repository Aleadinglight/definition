# Definition

Getting the definition of a word. Using `Node.js` and `Vue 2.0`. 

## Requirements

- Google Chrome
- Node.js Express Framework
- jQuery AJAX
- [Wordapi's API key](https://www.wordsapi.com/)

## How to use

```bash
git clone https://github.com/Aleadinglight/definition
git clone definition
npm install
npm start
```
